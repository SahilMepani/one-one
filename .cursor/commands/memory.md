## How to Store Notes

> **CRITICAL** — follow this on every note-storing request without exception.

When the user says "add a note", "store in memory", "remember this", or similar:

- Write to the most relevant file in `.cursor/` (rules, AGENTS.md, etc.) — this is the single source of truth across all AI tools
- **Notes must be project-agnostic by default** — write them in a way that applies across any project, not tied to this specific codebase, unless the user explicitly says otherwise
